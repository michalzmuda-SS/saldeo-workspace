# SaldeoSMART Agents (Vmodel-S) — marketplace

Marketplace z 3 modelami do Claude Code / VS Code. Wersje: ssa-product 0.8.0, par-analytics 0.8.0, dt-data 0.6.0.

## Instalacja
```
claude plugin marketplace add <ORG/repo lub sciezka-do-tego-folderu>
claude plugin install ssa-product@saldeosmart-agents
claude plugin install par-analytics@saldeosmart-agents
claude plugin install dt-data@saldeosmart-agents
```
W VS Code te same komendy w panelu Claude Code jako `/plugin ...`. Zespol instaluje tylko swoj model.
